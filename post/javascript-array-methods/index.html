配列の操作(forEach, filter, map, some, every)

October 10, 2016

Array.prototype.forEach()

forEach は、与えられた関数 （callback）を配列に含まれる各要素に対して一度ずつ呼び出す。

// 配列の要素の合計を計算する
let sum = 0;
const elms = [10, 20, 30, 40];
elms.forEach(elm => {
  sum += elm;
});

console.log(sum);
> 100

Array.prototype.filter()

filter は、与えられた callback 関数を配列の各要素に対して一度ずつ呼び出し、callback が真の値( true )を返したすべての値からなる新しい配列を生成する。

// 配列の要素が20で割り切れるものだけを抽出する
const before = [10, 20, 30, 40];
const after = before.filter(elm => {
  return ((elm % 20) === 0);
});

console.log(after);
> [20, 40]

Array.prototype.map()

map は、与えられた callback 関数を配列の各要素に対して順番に一度ずつ呼び出し、その結果から新しい配列を生成する。

// 配列の全ての要素に10を乗算した配列を返す
const before = [10, 20, 30, 40];
const after = before.map(elm => {
  return elm * 10;
});

console.log(before);
> [10, 20, 30, 40]
console.log(after);
> [100, 200, 300, 400]

Array.prototype.some()

some は、与えられた callback 関数を、配列に含まれる各要素に対して一度ずつ、callback が真の値( true )を返す要素が見つかるまで呼び出す。 真の値を返す要素が見つかると、some メソッドはただちに true を返します。

// 要素に20以上の値があればtrueを返す
const elms = [10, 20, 30, 40];
const retVal = elms.some(elm => {
 console.log(elm);
 return (elm >= 20);
});
> 10
> 20

console.log(retVal);
> true

Array.prototype.every()

every は、与えられた callback 関数を、配列に含まれる各要素に対して一度ずつ、callback が偽の値( false )を返す要素が見つかるまで呼び出す。

// 要素に一つでも20以下の値があればfalseを返す
const elms = [40, 30, 20, 10];
const retVal = elms.every(elm => {
 console.log(elm);
 return (elm > 20);
});
> 40
> 30
> 20

console.log(retVal);
> false

スプレッド演算子(spread operator)について

上記では配列に対しての操作を例に出しましたが、スプレッド演算子を使うことで配列以外のオブジェクトも同様に処理することが可能となります。

<div class="sample">
  <ul>
    <li>list1</li>
    <li>list2</li>
    <li>list3</li>
  </ul>
</div>

// リストのテキストのみを抽出する
const elms = document.querySelectorAll('.footer-link li');
const retVal = [...elms].map(elm => {
 return elm.textContent;
});

console.log(retVal);
> ["list1", "list2", "list3"]
