Hugo + Github Pagesでブログを公開してみた - Qiita

ブログを自分で構築したいという思いがずっとあったけれど、独自構文が多いらしい WordPress を勉強する気にあまりなれずにいたところ、
Jekyll や Hugo などを使って静的にブログを生成できることを知り、作ってみることにしました。
とりあえずはシンプルな hugo-zen というテーマを使ってみました。のちのちは、カスタマイズしたりテーマを作るなどしていきたいなあと思います。
